FAQ

Frequently asked questions before ordering.

Every project is quoted individually. Simply send approximate dimensions, the preferred shape and an inspiration image.

Does a loft-style table need black legs?+

No. The structure can also be finished in gold, white, anthracite or another selected RAL colour.

Is the steel base stable?+

Yes, provided its geometry, profiles and support points are matched to the dimensions and weight of the tabletop. Every structure is designed for a specific table.

Do you make extendable loft-style tables?+

Yes. We select the mechanism and base layout so the table remains stable and visually balanced when extended.

Which tabletop works with a black base?+

Black pairs well with light marble patterns, travertine, concrete and dark sintered stone with subtle veining.

Can I order the base only?+

The scope is agreed individually. We usually make the complete table so we can ensure the structure is properly matched to the tabletop.

Is a loft-style table suitable for a small living room?+

Yes. Visually lighter bases, rounded corners or an extendable version work particularly well in smaller interiors.
